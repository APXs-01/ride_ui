# RIDE — Pulse of the Ride

A hyper-premium, Awwwards-level scrollytelling landing page for RIDE Energy Drink.

## Stack

- **Vite + React** — build tooling
- **GSAP + Framer Motion** — animations
- **Canvas API** — scroll-driven image sequence (192 frames)
- **Inter** — typography

## Features

- 192-frame scroll-scrubbed canvas sequence
- 5 cinematic scroll phases with blur-in copy
- Glassmorphism navbar
- Live phase progress indicator
- Dark void aesthetic — `#030303` / `#C1FF00` / `#2E5BFF`
